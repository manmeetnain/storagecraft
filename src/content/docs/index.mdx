## What is StorageCraft?

StorageCraft is a free, open-source knowledge base for storage systems and AI infrastructure — from how NVMe works under the hood, to how KV-cache powers LLM inference.

## Sections

- **Core Concepts** — WAL, write amplification, copy-on-write, erasure coding
- **AI Infrastructure** — KV-cache, vLLM, Flash Attention, GPU memory
- **Simulators** — Interactive RAID, LSM tree, consistent hashing demos
- **Benchmarks** — Reproducible, Python-scripted benchmark data
