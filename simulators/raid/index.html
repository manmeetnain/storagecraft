How RAID-5 Works
RAID-5 stripes data across N disks with distributed parity. For 5 disks: Usable = (N-1) × disk_size
= 4 × disk_size = 80% Each stripe row has exactly 1 parity block, rotating which disk holds it: P(stripe) = D0 XOR D1 XOR D2 XOR D3 If any one disk fails, every lost block is recovered by XORing the remaining blocks in that stripe. This is why RAID-5 tolerates exactly 1 disk failure — a second failure loses data permanently.
